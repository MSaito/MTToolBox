template<typename T >
int MTToolBox::AlgorithmEquidsitribution< T >::get_all_equidist ( int  veq[])

vビット精度の均等分布次元を計算する。 v = 1 から bit_len までの均等分布次元を計算して、veq[] に入れる。返却値はv=1からbit_len までの均等分布次元の理論的上限との 差の総和である。

注意： AlgorithmEquidistribution をコンストラクトしてから、 get_all_dquidist() または、get_equidist() のどちらか一方しか 呼び出すことはできない。
